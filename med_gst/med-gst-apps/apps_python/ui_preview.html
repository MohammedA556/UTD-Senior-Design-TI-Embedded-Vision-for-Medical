Medical Vision UI Preview - post_process_ui_edit.py

1. Title Frame

Function: create_title_frame()
Shows the main title banner at the top of the application.
Colors: Blue (255,0,0) for main title, Red (0,0,255) for subtitle
Font: HERSHEY_SIMPLEX, size 1.2 and 1.0

2. Classification UI

Function: overlay_topN_classnames()
Displays top N recognized classes with confidence scores.
Background: Dark blue (5,11,120)
Title Color: Green (0,255,0)
Class Names: Yellow (255,255,0)

3. Object Detection UI

Function: overlay_bounding_box()
Shows bounding boxes around detected objects with labels.
Box Color: Dataset-specific RGB colors
Text Color: Auto-calculated based on background luminance
Box Thickness: 2 pixels

4. Keypoint Detection UI

Function: PostProcessKeypointDetection.__call__()
Displays skeleton keypoints and connections for pose estimation.
Bounding Box: Green (0,255,0)
Keypoints: Blue circles (255,0,0), radius 3px
Skeleton Lines: Blue (255,0,0), thickness 1px
